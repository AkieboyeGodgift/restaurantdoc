📌 Introduction

This package includes 3 single-page restaurant website templates built with HTML, TailwindCSS, JavaScript (Alpine.js), and Google Fonts. Each design is mobile responsive and tailored for fast customization and deployment.

⚙️ Installation & Usage

  1. Download and unzip the template files.
  2. Open any of the template folders.
  3. IMPORTANT: To use the template properly, you must run it on a live or demo server. Opening the HTML file directly in your browser may cause Alpine.js or some styles not to work correctly.
  4. Recommended ways to run a server:
    • VS Code + Live Server extension
    • Python: python3 -m http.server
    • Netlify Drop, Vercel, GitHub Pages, etc.
  5. Customize content, colors, and assets to match your brand.
